# SPC Pathfinder (v0 Demo)

This is a prototype of the SPC Member Directory, built using **v0** and the Vercel AI stack to demonstrate agentic product velocity. Key features include a **High-Signal VC Index** organizing investors by firm and thesis, an **Agentic Drafter** that auto-generates context-aware email requests, and a **Request CRM** to track the full lifecycle of a warm intro. Built with Next.js/Tailwind; all data is mocked to simulate realistic relationship context.
